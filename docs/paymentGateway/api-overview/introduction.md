# Introduction

Welcome to the Z.ro Bank Payments Gateway API documentation. Our API provides a comprehensive set of
endpoints to enable developers to build powerful applications that interact with our technology.

This documentation provides all the necessary information to help you integrate with our API seamlessly.

## API ID and API Key Setup

Once your `API ID` and `API Key` are created, you are ready to start the integration.

- If you have not registered your company yet, please
  access: [Register to Z.ro Bank Payments Gateway API](https://docs.google.com/forms/d/e/1FAIpQLSdMHWF1sZm7jjSsKiYGZIcd7sN2vCwx_OH5Eh3W1X8n7wS6ug/viewform)

- If you have already registered your company, but your credentials were not provided yet, please contact
  our [support team](/overview/support).

> Note: Never share your `API ID` and `API Key` to anyone. If you have accidentally shared, please contact
> our [support team](/overview/support) immediately.
